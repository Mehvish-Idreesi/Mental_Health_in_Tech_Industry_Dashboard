This Power BI dashboard analyzes how mental health issues are perceived and supported in the tech industry. Using a dataset of 1,245 employees across various countries and company sizes, it explores key factors such as employer support, stress levels, job satisfaction, and access to mental health resources.

The dashboard highlights how workplace conditions, remote work patterns, and company policies influence employee well-being. Interactive slicers for continent, gender, company size, and work mode allow deeper insights into different segments of the tech workforce.

Through KPIs, charts, and correlation visuals, this dashboard aims to uncover trends in mental health awareness and support, helping organizations recognize the importance of fostering a healthier and more open work culture.
